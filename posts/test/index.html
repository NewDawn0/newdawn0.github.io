Listing all installed packages:

There are often multiple ways to archieve something with shin. Most commonly there you have the option to type the full word or just the first letter(s) but starting with a minus. Anyway here are the two ways to uninstall a package with Shin

Example:

sudo shin list

sudo shin list-packages

sudo shin -l

sudo shin list-package-managers

Installing multiple packages:

Installing multiple packages with shin is quite simple. All you need to do is enter more package names with the methods described above. Firstly we’ll go over installing multiple wihout a specific package manager just like in the first part of the post, later we will look at installing multiple packages with the same package manager

Installing Multiple packages without a specific package manager:

When installing multiple packages this way the package manager doesn’t matter because shin will just loop through all the packages and install the first match no matter the package manager This method is usually the most suited method for installing multiple packages.

Basic syntax:

sudo shin install <package name> <package name> <package name>

sudo shin -i <package name> <package name> <package name>

Example:

sudo shin install yaf didyoumean alacritty

sudo shin -i yaf didyoumean alacritty

Output:

Installing Multiple packages with the same package manager:

When installing multiple packages with the same package manager, be aware that shin will only search that package manager. Meaning if a package isn’t available there it will just say the package isn’t found.

Basic syntax:

sudo shin pkgman-install <package manager> <package name> <package name> <package name>

sudo shin -pi <package manager> <package name> <package name> <package name>

sudo shin -ip <package manager> <package name> <package name> <package name>

Example:

sudo shin pkgman-install brew alacritty pfetch hyper

sudo shin -pi brew alacritty pfetch hyper

sudo shin -ip brew alacritty pfetch hyper
